resize

将字符串的大小,注意这里是字符串长度不是字符串容量，要注意区分，调整为n个字符的长度。如果n小于当前字符串长度，则将当前值缩短为其前n个字符，删除第n个字符以外的字符。
如果n大于当前字符串长度，则通过在最后插入所需数量的字符来扩展当前内容以达到n的大小。 如果指定了c，则新元素将初始化为c的副本，否则，填充空字符。
