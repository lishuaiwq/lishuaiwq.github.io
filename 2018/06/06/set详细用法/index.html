set详细用法

发表于 2018-06-06 | 分类于 数据结构随笔 | 热度: ℃
字数统计: 515 | 阅读时长 ≈ 2

此篇博客主要介绍的是set里面的一些重点常用的用法剖析，不涉及set自身底层的实现。当然只是挑重点说，所以说不够全面，详细希望大家查阅http://www.cplusplus.com/reference/set/set/insert/

  • 首先给出set的类的模板接口

Insert

通过插入新元素来扩展容器，通过插入元素的数量有效地增加容器大小。由于集合中的元素是唯一的，所以插入操作会检查每个插入的元素是否与容器中已存在的元素相同，如果是，则不插入该元素，并将迭代器返回到此现有元素（如果该函数返回一个值）。对于允许重复元素的类似容器，请参阅multiset。在内部，设置容器将按照比较对象指定的标准对所有元素进行排序。 这些元素总是按照这个顺序插入到相应的位置。

插入成功了返回一个pair类型，如果插入成功的话返回插入元素的位置和true，如果不成功(一般来说就是重复的元素，那么返回相同元素的位置和false)

Find

在set中查找指定值，如果找到了则返回当前元素的位置，如果没有找到则将迭代器设置为end()
