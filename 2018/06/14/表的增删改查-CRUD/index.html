删除

删除表中的数据

  • 删除整个表中的内容还有另外的一个命令truncate

上述两种删除整表的区别:

  • 效果一样，truncate速度快
  • delete可以带where条件，删除更加灵活
  • delete可以返回被删除的记录数，而truncate返回0
  • 推荐使用delete
    delete使用细节：
  • 如果不适用where子句，将删除整个表中所有数据
  • delete语句不能删除某一列的值（可以用update置null）
  • 使用delete语句仅删除记录，不删除表本身（drop table

select的order by子句

  • order by 指定排序的列，排序的列可以使表中的列名，也可以是select语句后指定的别名
    • asc升序(默认)，desc降序
    • order by 子句应该位于select语句的结尾

  • count(列名) 返回某一列的行数

  • sum()统计总和

    注意：sum仅对数值起作用，否则结果无意义

  • max()统计最大的

  • min()统计最小的

  • avg()返回满足where条件的一列平均值

    mysql> select avg(math+english+chinese) from student;
    format(math,2);//小数位保留两位;

  • group by:按照某个字段分类，即将某个字段值相同的分为一组

select avg(sal) , deptno from EMP group by deptno;//按部门分类，并且求每个分类部门的平均工资
